# Fraud Detection ML Application

An end-to-end Machine Learning project designed to detect fraudulent transactions using a structured, production-ready pipeline and deployed as an interactive Streamlit web application.

## Objective

To build a scalable and interpretable fraud detection system capable of identifying high-risk transactions with strong predictive performance.

## Model Performance

- **Accuracy: 94%**
- Optimized through hyperparameter tuning
- Built with a fully integrated preprocessing + modeling pipeline
